# StellasoraMaster 使用者指南

返回首页： [README.md](README.md)

本指南面向日常使用者，不需要了解代码即可上手。

## 1. 这是什么

StellasoraMaster 是一个《星塔旅人》自动化助手，提供：

- 日常任务自动执行
- 自动爬塔（可选快速/标准）
- 实时画面监控
- 图形化设置（ADB 路径、端口、语言、截图方式）

---

## 2. 使用前准备

请先确保以下条件：

1. 已安装并可正常打开模拟器（MuMu12 / 雷电 / 自定义）
2. 已安装游戏并可进入主界面
3. 已确认对应模拟器的 adb.exe 路径
4. 电脑已安装 Python（源码运行时需要）

常见端口：

- MuMu12：16384
- 雷电：5555

---

## 3. 启动方式

在项目目录 Game_screenshotReed_Autowork 下执行：

方式 A：

```bash
python launcher.py
```

方式 B：

```bash
python run_app.py
```

启动成功后会自动打开浏览器页面（默认地址通常是 `http://127.0.0.1:5000/`）。

---

## 4. 首次设置（非常重要）

打开页面后，先进入“设置”标签页：

1. 选择模拟器类型
2. 填写 ADB 路径（adb.exe 完整路径）
3. 填写 ADB 端口
4. 选择截图方式
   - 兼容模式 PNG：更稳，速度偏慢
   - 高速模式 RAW：更快，推荐优先尝试
5. 点击“测试连接”
6. 点击“保存设置”

建议：首次使用优先保证“测试连接成功”后再执行任务。

---

## 5. 日常使用流程

## 5.1 任务执行

在“任务执行”页：

1. 勾选“日常任务流程”或“自动爬塔”（可同时勾选）
2. 如果勾选日常，可进一步勾选子任务
3. 如果勾选爬塔，设置属性、模式、次数
4. 点击“开始执行”
5. 执行中可使用：暂停 / 继续 / 停止

### 日常子任务说明（简要）

- 主页面角色互动
- 商店随机奖励
- 委托派遣
- 好友管理（可设置添加数量）
- 赠送礼物
- 邀约（支持 5 角色）
- 秘纹升级
- 旅人升级
- 领取日常奖励

### 爬塔参数说明

- 属性：光地 / 水风 / 火暗
- 模式：
  - 快速：更省时间
  - 标准：完整流程（包含更多步骤）
- 次数：
  - 0 表示不限（通常跑到周常达标或资源不足）

## 5.2 实时监控

在“实时监控”页：

- 点击“开启监控”查看实时画面
- 点击“暂停监控”暂停画面流
- 点击“刷新画面”重新拉流

## 5.3 日志导出

在任务页右侧“服务端日志”中可点击“导出”，便于排查问题。

---

## 6. 常见问题

## 6.1 提示连接失败

请依次检查：

1. ADB 路径是否填写正确
2. 端口是否与模拟器一致
3. 模拟器是否已启动并进入系统
4. 设置页“测试连接”是否成功

## 6.2 任务点击不准 / 无反应

建议：

- 确认游戏窗口分辨率、缩放比例正常
- 切换截图方式（RAW 与 PNG 互换测试）
- 确认当前服务器语言与模板语言一致（简中/繁中）

## 6.3 监控画面卡住

可尝试：

1. 先暂停监控再开启
2. 点击“刷新画面”
3. 切换截图模式并保存

## 6.4 更新失败

常见原因是网络或下载地址不可达。可稍后重试，或导出日志后让维护者协助排查。

---

## 7. 安全与使用建议

- 初次使用建议先小范围测试（只勾选 1~2 个子任务）
- 运行时尽量不要手动频繁切换游戏界面
- 如果要长期挂机，建议先确认暂停/停止按钮可正常响应
- 重要账号建议先在小号验证流程

---

## 8. 给维护者反馈时请提供

为了更快定位问题，建议提供：

1. 复现步骤
2. 当前配置截图（可打码）
3. 导出的日志文件
4. 问题发生时的监控画面截图

完成以上信息后，维护者通常能更快给出修复方案。
