# StellasoraMaster

星塔旅人自动化辅助工具（Python + Flask + Vue）。

## 快速入口

- 开发文档： [开发文档.md](开发文档.md)
- 使用者指南： [用户使用指南.md](用户使用指南.md)

## 核心功能

- 自动执行日常任务（支持子任务勾选）
- 自动爬塔（属性/模式/次数可配置）
- 实时监控画面（MJPEG）
- 设置管理（ADB 路径、端口、语言、截图方式）
- 日志查看与导出、在线更新
